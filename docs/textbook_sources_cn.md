# 中国小学教材与教辅资料来源清单（首版）

> 说明：你提到“搜寻所有中国小学教材以及教辅资料”。  
> 互联网上不存在一个稳定、完整、永久可下载的“全量公开全集”入口，且教材/教辅存在版权与授权限制。  
> 本清单先整理 **官方或权威来源** 作为项目数据接入基线，后续可持续扩展。

## A. 国家级平台（优先）

1. 国家中小学智慧教育平台（教育部）
   - 入口: https://basic.smartedu.cn/
   - 特点: 覆盖多出版社、多版本教材与课程资源，是最重要的国家级来源。

2. 教育部官网（政策与资源通知）
   - 入口: http://www.moe.gov.cn/
   - 示例: 全国中小学图书馆（室）推荐书目等通知与资源公告。

## B. 主要教材出版社（官方）

1. 人民教育出版社（PEP）
   - 官网: https://www.pep.com.cn/
   - 教学资源: https://www.pep.com.cn/jxzy/
   - 数字教材相关: https://www.pep.com.cn/products/sz/jjfa/rjszjc/

2. 江苏凤凰教育出版社
   - 官网: https://www.1088.com.cn/

3. 外语教学与研究出版社
   - 官网: https://www.fltrp.com/

4. 华东师范大学出版社（教辅分社）
   - 官网: https://jiaofu.ecnupress.com.cn/

## C. 教辅与扩展学习资料来源建议

1. 各教材出版社“配套资源/教辅”栏目（优先官方）
2. 各地教育局、教研室公开资源平台
3. 国家中小学智慧教育平台的课后服务、家庭教育、课程教学资源板块

## D. 数据接入策略建议（用于产品）

1. **资源目录先行**：先存“版本-年级-学科-册次-课时”元数据，不直接抓取全文内容。
2. **授权后取内容**：通过 API、授权接口或学校侧正版资源导入。
3. **统一章节编码**：建立 `publisher/version/grade/subject/unit/lesson` 标准主键。
4. **资源分级缓存**：教材目录（公开）与题目内容（授权）分层存储。

## E. 风险与合规

1. 教材和教辅通常受版权保护，禁止未授权批量抓取与分发。
2. 涉及学生作业与错题记录属于个人学习数据，需最小化采集、明确告知、可删除。
3. 若接入拍照识题和大模型，需额外评估图像与文本数据的跨境/第三方处理合规。
